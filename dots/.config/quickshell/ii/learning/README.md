# Quickshell Learning Project

Open this project with:

```sh
qs -p learning/StartHere.qml
```

This is a beginner project for people who have never coded before.

## What this teaches

1. How a Quickshell window is built.
2. How text changes when a value changes.
3. How a button can change a value.
4. How to make a tiny light/dark toggle inside one file.

## Start Here

Open `learning/StartHere.qml` and look for the comments marked:

- `EDIT ME`
- `TRY THIS`
- `WHAT THIS MEANS`

Those comments are written for non-programmers.

## Safe way to learn

Change one thing at a time.

Good first experiments:

1. Change the title text.
2. Change the greeting text.
3. Change the colors.
4. Change the number in the counter.

If something breaks, close the window and open it again with the command above.
